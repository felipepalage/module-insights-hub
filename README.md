# Module Insights Hub

Dashboard de Monitoramento de Módulos.

## Project info

Este projeto é um hub de insights para monitoramento de módulos, desenvolvido para fornecer uma visão clara e objetiva das operações.

## Tech Stack

Este projeto foi construído com:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Desenvolvimento Local

Siga estes passos para rodar o projeto localmente:

1. Instale as dependências:
```sh
npm install
```

2. Inicie o servidor de desenvolvimento:
```sh
npm run dev
```
